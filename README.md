# ReadLay  
Bet on yourself. Read more. Learn, apply, and grow.  

ReadLay is an iOS app that gamifies reading by turning your goals into bets. Inspired by sports parlays, it helps you stay accountable, track your progress, and reflect on what you read.  

---

## Features  

- Reading Bets: Set single or parlay-style goals across multiple books.  
- Reading Timer: Log sessions in real time and keep track of your consistency.  
- Journaling: Add notes and reflections after each session to reinforce learning.  
- Progress Tracking: Visualize daily and overall reading progress.  
- Book Data Integration: Pull real book data (titles, covers, page counts) from OpenLibrary and Google Books APIs.  
- Modern UI/UX: Built with SwiftUI for a clean, smooth user experience.  
- Performance and Testability: Continuously improving speed, structure, and user flow.  

---

## Tech Stack  

- Mobile Development: Swift, SwiftUI, UIKit  
- Concurrency: Swift Concurrency (async/await) for networking and background tasks  
- Architecture: MVVM (with elements of MVC)  
- Persistence: Core Data for saving books, sessions, and journal entries  
- Networking: REST APIs (OpenLibrary, Google Books), JSON decoding  
- Backend/Cloud: Firebase (for authentication and data sync, if applicable)  
- Version Control: Git, GitHub
